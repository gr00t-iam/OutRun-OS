THE OUTRUN OS PROJECT MANIFESTO =============================== The desktop computer model is broken. Modern operating systems are bogged down by structural technical debt, aggressive tracking scripts, rigid corporate stores, and forced update restarts that treat users like commodities instead of owners. We are building a clean alternative: an OS that honors performance, visual beauty, and personal autonomy. 1. ARCHITECTURAL SANITY via TOUGH SEPARATION No more blue screens or frozen desktops caused by buggy drivers. By shifting drivers and file systems out of the microkernel and into isolated spaces using Rust, C++, and Assembly, a peripheral crash remains entirely unnoticeable to the system. 2. UNRESTRICTED HARDWARE CONTROL We reject the trend of locking down hardware components behind endless corporate prompts. Applications on this system have direct, zero-copy, memory-mapped access to cameras, microphones, and game controllers. Maximum hardware output is delivered directly to you without systemic gatekeeping. 3. THE CHRONOLOGICAL WORKSPACE Files belong to timelines, not hidden nested folders. The infinite canvas visualizes your workflow chronologically, using localized edge AI computing to automatically index documents, streams, and chats safely within your own machine. 4. CAMERA SPATIAL CONTROL Input devices must work together. Our system treats traditional keyboards, mouse inputs, and real-time camera gesture tracking as equal partners, enabling seamless, fluid navigation without performance lag. 5. OPEN MAINSTREAM CHANNELS The Outrun Marketplace preserves uniform UI styling and rigorous open source tools completely tax-free. At the exact same time, decentralized side-loading remains a native, uncompromised right for developers and consumers alike. This is computing designed around speed, privacy, and absolute user freedom.
